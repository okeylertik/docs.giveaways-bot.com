# Introduction
 
Welcome to the official Documentation site of the GiveAways bot! We hope this Documentation is good and you can use it very well :D

## What is GiveAways?

A **powerful** and **advanced** discord GiveAway bot.

## What is our goal?

Our goal is to make the best GiveAway bot for discord. We want to make it as easy as possible for you to create GiveAways and to make it as easy as possible for you to use the bot.

## What is the prefix?
We have entirely changed / switched to slash commands. This means that you no longer need a prefix. You can now use the commands directly in a chat with the ``/``.

## Where is the support server?
You can find our support server here: [discord.gg/giveawaysbot](https://discord.gg/giveawaysbot)
